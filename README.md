# Cybersecurity EDA - UNSW-NB15 Dataset

Exploratory Data Analysis of 2.5M+ network traffic records from the UNSW-NB15 intrusion detection dataset using Python/Pandas.

### **Key Findings**
- **Class imbalance**: 87% Normal traffic vs 13% Attacks - shows why accuracy alone fails for intrusion detection
- **Top threats**: Exploits, Fuzzers, and DoS attacks dominate the dataset
- **Protocol patterns**: TCP carries majority of attacks vs UDP/ICMP

### **Visualizations**
1. Attack vs Normal distribution - severe class imbalance
2. Top 10 attack categories by frequency  
3. Protocol distribution across attack types
